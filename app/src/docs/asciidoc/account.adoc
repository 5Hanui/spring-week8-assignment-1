[[Account]]
= Account
회원 정보 관리 API

=== 공통 사항

회원 관련 API는 다음과 같은 공통 사항을 지켜야 한다.

=== Request Header

|===
|Name |Description

|Authorization
|API를 사용하기 위한 인증 토큰
|===

=== 상품 Response Body
|===
|Name |Description

|id
|회원 식별자 정보

|name
|회원 이름

|email
|회원의 이메일 주소
|===
