== 고양이 장난감 상품 API

== 공통 사항

고양이 장난감 상품 관련 API는 다음과 같은 공통 사항을 지켜야 한다.

=== Request Header

|===
|Name |Description

|Authorization
|API를 사용하기 위한 인증 토큰
|===

=== 상품 Response Body

|===
|Name |Description

|id
|상품 식별자 정보

|name
|상품명

|maker
|상품 제조사

|price
|상품 가격

|imageUrl
|상품 이미지 주소
|===
