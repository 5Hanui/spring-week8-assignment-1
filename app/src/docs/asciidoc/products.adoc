[[POST-products]]
=== 상품 등록하기
새로운 상품을 등록하고 등록 된 상품 정보를 반환 받는 API입니다. +
새로운 상품을 등록하기 위해서는 `인증 토큰` 이 필요합니다.

TIP: 인증 토큰(Access-Token)은 사용자가 로그인 시 발급받을 수 있습니다. +
만일 로그인이 만료 되었거나 분실 되었다면, 새롭게 로그인하여 인증 토큰을 발급받으실 수 있습니다.


==== Request
===== URL
[source]
POST /products/ HTTP/1.1
Accept: application/json;charset=UTF-8
Authorization: Bearer {Access-Token}
Host: localhost:8080

상품의 정보를 수정하기 위해서는 `인증 토큰` 이 필요합니다.
