[[Session]]
== 세션 정보 관리하기

이 섹션은 인증 토큰을 발급받기 위한 기능 API에 대해 안내합니다.

[[Session-POST]]
=== 인증 토큰 발급받기
인증 토큰을 발급해주는 기능을 제공하는 API입니다. +
인증 토큰을 발급받기 위해서는 회원의 이메일 및 비밀번호 정보를 필요로 합니다.

==== Request
===== URL
[source]
POST /session HTTP/1.1
Content-Type: application/json;charset=UTF-8
Host: localhost:8080
{
    "email":{email},
    "password":{password}
}

===== Response
[cols="3,1,5,1a"]
|===
|Name |Type |Description |Required

|accessToken
|String
|인증 토큰 정보
|O

|===
